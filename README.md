# amigo-secreto
<h1>Amigo Secreto</h1>

En este desafío se desarrollará una aplicación interactiva que permitirá organizar un sorteo del “amigo secreto”. El funcionamiento será sencillo y estará basado en cuatro funcionalidades principales:

Agregar nombres: El usuario podrá escribir el nombre de un amigo en un campo de texto. Al hacer clic en el botón “Adicionar”, el nombre se añadirá automáticamente a una lista visible en la página.

Validar entrada: Si el usuario intenta adicionar un nombre dejando el campo vacío, el programa no lo permitirá y mostrará una alerta solicitando que se ingrese un nombre válido.

Visualizar la lista: Todos los nombres ingresados aparecerán en una lista debajo del campo de entrada, de manera que el usuario pueda ver en todo momento quiénes están participando en el sorteo.

Sorteo aleatorio: Una vez que se haya completado la lista, el usuario podrá hacer clic en el botón “Sortear Amigo”, el cual seleccionará de manera aleatoria uno de los nombres de la lista y mostrará el resultado directamente en pantalla.
